Class: PDFDoc

PDFNet. PDFDoc

new PDFDoc()

PDFDoc is a high-level class describing a single PDF (Portable Document Format) document. Most applications using PDFNet will use this class to open existing PDF documents, or to create new PDF documents from scratch. The class offers a number of entry points into the document. For example, - To access pages use pdfdoc.GetPageIterator() or pdfdoc.PageFind(page_num). - To access form fields use pdfdoc.GetFieldIterator() or pdfdoc.FieldFind(name). - To access document's meta-data use pdfdoc.GetDocInfo(). - To access the outline tree use pdfdoc.GetFirstBookmark(). - To access low-level Document Catalog use pdfdoc.GetRoot(). ... The class also offers utility methods to slit and merge PDF pages, to create new pages, to flatten forms, to change security settings, etc.
